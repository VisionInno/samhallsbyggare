Bakom kulisserna

Byggd på en dag. På riktigt.

Samhällsbyggarkartan är ett experiment med en enkel fråga: hur långt kommer man idag med öppna data och AI-assisterad utveckling? Svaret visade sig vara: längre än de flesta tror.

Verktyget

Hela sajten är byggd i dialog med Claude Code — Anthropics AI-verktyg för utveckling. Människan beskrev vad som skulle byggas och fattade besluten; AI:n skrev koden, verifierade datakällorna och satte upp driften i Azure. Ingen i teamet är programmerare.

Datat

Nio öppna källor från svenska myndigheter och OpenStreetMap-ekosystemet, utan en enda API-nyckel. Kartlagren hämtas direkt från källornas egna tjänster — samma data som proffsens GIS-system använder. Se hela listan.

Tekniken

Medvetet enkel: ren HTML, CSS och JavaScript utan byggprocess. Leaflet och MapLibre GL för kartan, Azure Static Web Apps (gratisnivån!) för driften, GitHub Actions för automatisk deploy vid varje ändring. Den enda "backend" som finns är en minimal vidarebefordran för de myndighetstjänster som inte tillåter anrop direkt från webbläsaren.

Koden är öppen

Varför?

Poängen för samhällsbyggare

Sveriges myndigheter har gjort ett fantastiskt jobb med att öppna sina geodata. Men datat gör ingen nytta förrän det landar i händerna på dem som fattar besluten — exploatörer, planerare, byggare, mäklare, kommuner. Tröskeln är inte längre pengar eller teknik. Den här sajten byggdes på en dag. Vad skulle din organisation kunna bygga på en vecka?
